## Latam Remote Dev -  Backend 
## Description
Job board designed for software developers

## Api documentation
[Link text]((https://latam-remote-dev-2-dev-kgpt.1.us-1.fl0.io/docs)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn  start:dev

# production mode
$ yarn  start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
