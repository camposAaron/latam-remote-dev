#LATAM REMOTE DEV - API 
## Description
API para la app LATAM REMOTE DEV 
bolsa de trabajos hecha para desarrolladores de software.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn  start:dev

# production mode
$ yarn  start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
